# Marketplace Search & Ranking Analysis (Olist Dataset)

## Overview

This project simulates a Search & Ranking analytics pipeline for an e-commerce marketplace using the Brazilian Olist dataset.

The goal is to engineer ranking signals and build interactive dashboards to evaluate seller performance, product quality, delivery reliability, and geographic marketplace health.

## Tools Used

Python:
- pandas
- feature engineering
- ranking signal construction

Tableau:
- interactive dashboards
- ranking signal visualization

## Ranking Signals Engineered

Seller-level:
- seller_quality_score
- seller_late_delivery_rate
- seller_avg_review_score

Product-level:
- product_quality_score

Category-level:
- category_quality_score

Geographic-level:
- geo_quality_score

Customer-level:
- customer_value_score
